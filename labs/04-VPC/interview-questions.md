# Interview Questions

## What is Amazon VPC?

Amazon VPC allows users to create isolated virtual networks inside AWS.

---

## Difference between Security Group and Network ACL?

Security Groups are stateful.

Network ACLs are stateless.

---

## What is an Internet Gateway?

It provides Internet connectivity between a VPC and the Internet.

---

## Why use public and private subnets?

Public subnets host Internet-facing resources.

Private subnets host internal services such as databases.

---

## What is a Route Table?

A Route Table determines how network traffic leaves a subnet.

---

## Why enable Auto Assign Public IP?

So instances in public subnets receive a public IPv4 automatically.
